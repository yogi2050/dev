Java.util.Locale.getISOLanguages() Method

Advertisements
Previous Page
Next Page

Description

The java.util.Locale.getISOLanguages() method returns a list of all 2-letter language codes defined in ISO 639. Can be used to create Locales.

Declaration

Following is the declaration for java.util.Locale.getISOLanguages() method

public static String[] getISOLanguages()

Parameters

  • NA

Return Value

This method does not return a value.

Exception

  • NA

Example

The following example shows the usage of java.util.Locale.getISOLanguages() method.

package com.tutorialspoint;

import java.util.*;

public class LocaleDemo {

   public static void main(String[] args) {

      // create a new locale
      Locale locale = new Locale("en", "US", "WIN");

      // print locale
      System.out.println("Locale:" + locale);

      // get ISO languages
      String[] languages = Locale.getISOLanguages();

      // print countries
      System.out.println("Languages are:");
      for (int i = 0; i < languages.length; i++) {
         System.out.println(i + ":" + languages[i]);
      }
   }
}
