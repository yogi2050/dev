You have leanred about WSDL, now the next step is to learn about SOAP, UDDI and Web Services.

Web Services

Web services are open standard ( XML, SOAP, HTTP etc.) based Web applications that interact with other web applications for the purpose of exchanging data.

To learn more about Web Services visit Web Services Tutorial

UDDI

UDDI is an XML-based standard for describing, publishing, and finding Web services.

To learn more about UDDI visit UDDI Tutorial

SOAP

SOAP is a simple XML-based protocol that allows applications to exchange information over HTTP.

To learn more about SOAP visit SOAP Tutorial
