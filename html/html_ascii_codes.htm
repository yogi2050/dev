HTML ASCII Codes



Advertisements

There are 2**7 = 128 printable characters which can be represented by different 7-BIT ASCII codes. Another set of characters are not for HTML representation but they are deviced to control hardware.

Following tables list down all the 7-BIT ASCII codes and thier equivalent HTML Entity Codes.

If you want to see equivalent HEX, OCT and extended set of ASCII codes then check next chapter.
