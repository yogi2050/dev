RADIUS Attributes Format



Advertisements

RADIUS Attribute will consist of following three parts

  • Type: 1 Octet, identifies various types of attribute. It will be an attribute code listed below.

  • Length: 1 Octet, length of the attribute including Type

  • Value: 0 or more Octets, contains information specific to attribute.
